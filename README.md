# Weather App

Next.js 14 weather app with server-side API routes (API key hidden).

## Features
- City search suggestions
- Current location weather
- Forecast data (OpenWeather)
- React Query caching
- Tailwind UI

## Tech Stack
- Next.js (App Router)
- TypeScript
- TailwindCSS
- @tanstack/react-query
- Jotai

## Environment Variables
Create `.env.local`:

OPENWEATHER_API_KEY=your_key

## Run locally
npm install
npm run dev

## Build
npm run build
npm start
